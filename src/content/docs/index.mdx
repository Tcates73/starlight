## Send your first email in 60 seconds

```bash
curl https://api.sendflow.io/v1/send \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "customer@example.com",
    "from": "you@yourdomain.com",
    "subject": "Welcome to our platform!",
    "html": "<h1>Thanks for signing up!</h1>"
  }'
```

## Quick Integration Examples

### Node.js
```javascript
const sendflow = require('@sendflow/node');
sendflow.apiKey = 'your_api_key';

await sendflow.send({
  to: 'user@example.com',
  from: 'hello@yourdomain.com',
  subject: 'Welcome!',
  html: '<h1>Thanks for signing up!</h1>'
});
```

### Python
```python
import sendflow

sendflow.api_key = 'your_api_key'

sendflow.send(
  to='user@example.com',
  from_email='hello@yourdomain.com',
  subject='Welcome!',
  html='<h1>Thanks for signing up!</h1>'
)
```

### PHP
```php
$sendflow = new SendFlow\Client('your_api_key');

$sendflow->send([
  'to' => 'user@example.com',
  'from' => 'hello@yourdomain.com',
  'subject' => 'Welcome!',
  'html' => '<h1>Thanks for signing up!</h1>'
]);
```
